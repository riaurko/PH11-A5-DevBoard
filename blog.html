What are the different ways to select an element in the DOM?

There are 5 different ways. They are:
getElementsByTagName(tagName: string) which returns a HTMLCollection of all the elements based on a specific Tag.
getElementById(id: string) which returns a single element based on a specific Id.
getElementsByClassName(class: string) which returns a HTMLCollection of all the elements based on a specific Class.
querySelector(cssQuery: string) which returns a signle element based on a specific CSS Query.
querySelectorAll(cssQuery: string) which returns a NodeList of all the elements based on a specific CSS Query.

What is the difference between innerHTML, innerText and textContent?

innerHTML returns the HTML Nodes inside the target element, while innerText and textContent only returns the text (text nodes, not elements/attribute nodes) inside the target element. But there are difference between innerText and textContent. innerText only returns the text that are visible in webpage, meaning that are not hidden (like not have display: none;), while textContent returns all texts including that are hidden.

What is event delegation in DOM?

Event Delegation is a concept where you add a single EventListener to the target parent element instead of adding event listeners to all the childs. Like I used it in every Task Card in the Home Page to track the 'Complete' Button.

What is event bubbling in DOM?

Event Bubbling is a process where the event starts from the target element and then bubbles up to its parent element until it reaches the root/html element (document)

How do you create, add, and remove elements using JavaScript?

We can create an element using the document.createElement(tagName: string) method. And can add the element inside an element as a child using parentElement.appendChild(element: string) method, but note that it appends the child at the last. And can remove a child using parentElement.appendChild(child: string) method.
